## How to Create Screenshots

1. Run the app in dev mode: `npm run dev`
2. Use browser DevTools to set viewport size
3. Take screenshots using browser tools or OS screenshot
4. Save as PNG in this directory

## Screenshot Requirements
- Use actual app views with realistic data
- Hide any sensitive/personal information
- Use consistent styling/theme
- Ensure good contrast and readability
